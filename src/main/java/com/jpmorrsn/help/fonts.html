Fonts

DrawFBP uses two fonts: general and fixed size: the general is used for all text, while the fixed size is used for names of files.

Fonts can be changed either by selecting File/Change Fonts or by clicking on the button at the top of the display area. First the "general" fonts on your machine will be displayed, together with sample text in English and Chinese, allowing one to be selected; then the "fixed" fonts will be displayed for selection, in the same format.
